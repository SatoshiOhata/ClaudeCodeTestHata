# SEO監査スキル

HTMLファイルのSEO要素を監査し、改善提案を行う。

## チェック項目と基準

### 1. タイトルタグ
- 30〜60文字（日本語）が適正
- ターゲットキーワードを含む
- ブランド名を含む
- 評価：60文字以内で具体的 → ✅ / 短すぎ or 長すぎ → ⚠️ / 未設定 → ❌

### 2. メタディスクリプション
- 80〜120文字（日本語）が適正
- ページの内容を要約
- 行動喚起を含む
- 評価：設定済みで適切 → ✅ / 改善の余地あり → ⚠️ / 未設定 → ❌

### 3. 見出し構造
- H1は1つのみ
- H1 → H2 → H3の階層が正しい
- 見出しにキーワードを含む
- 評価：階層正しい → ✅ / 一部不備 → ⚠️ / H1なし or 複数 → ❌

### 4. OGPタグ
- og:title, og:description, og:image, og:url が設定されている
- Twitter Card タグも設定されている
- 評価：全て設定 → ✅ / 一部欠落 → ⚠️ / 未設定 → ❌

### 5. 構造化データ
- JSON-LD形式のSchema.orgマークアップ
- ページの種類に適したtypeを使用
- 評価：設定済み → ✅ / 不完全 → ⚠️ / 未設定 → ❌

### 6. 画像のalt属性
- すべてのimgタグにalt属性がある
- altが具体的で内容を説明している
- 評価：全て設定 → ✅ / 一部欠落 → ⚠️ / 大量に欠落 → ❌

### 7. canonical URL
- link rel="canonical" が設定されている
- 評価：設定済み → ✅ / 未設定 → ❌
