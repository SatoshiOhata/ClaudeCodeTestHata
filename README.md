# ClaudeCodeTestHata

Claude Code の動作テスト用リポジトリです。

## 概要

このリポジトリは、[Claude Code](https://docs.anthropic.com/en/docs/claude-code)（Anthropic公式CLI）の機能検証・テストを目的として作成されました。

## 環境

- **リモートリポジトリ**: [GitHub - SatoshiOhata/ClaudeCodeTestHata](https://github.com/SatoshiOhata/ClaudeCodeTestHata)
- **デフォルトブランチ**: `main`
- **Git LFS**: 有効

## 使い方

1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/SatoshiOhata/ClaudeCodeTestHata.git
   ```
2. Claude Code を使って開発・テストを行います。
